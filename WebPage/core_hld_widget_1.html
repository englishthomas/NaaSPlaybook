Mobile Core HLD - Capacity Planning

The Capacity Planning Widget's objective is to ease the NaaS Operator the calculations of the User & Control Planes capacity.

It includes:

  • The estimation of throughput of each interface considering the amount of data generated by the user and their signaling as well.
  • The quantification of the necessary equipment to cope with the Control and User Plane previously calculated.

Please fill the blanks with information needed then clic the "Calculate" button.

Basics for this wizard are addressed on the Mobile Core HLD Module.
