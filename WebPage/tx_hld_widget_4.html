Tx Microwave

This widget calculates the Link Budget for a Microwave Link based on the link's general characteristics.

In order to use this widget, perform the following steps:

  • Introduce the MW link characteristics in the Inputs section, according to the given units.
  • Click on the "Calculate" button to generate the MW Link Budget.
  • The MW Link Budget parameters can be found in the Results section.

Basics for this widget are addressed on the TX HLD Module.
