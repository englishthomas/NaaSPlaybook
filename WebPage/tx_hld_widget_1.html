Tx - Capacity Forecast

This widget estimates the Link Capacity for Last-mile and Aggregation Links based on Access network general parameters.

In order to use this widget, perform the following steps:

  • Introduce the Access network characteristics in the Inputs section, according to the given units.
  • Click on the "Calculate" button to estimate the Last-mile Link Capacity Calculation.
  • The Link Capacity parameters can be found in the Results section.
  • To calculate the Link Capacity for an aggregation link, introduce the number of aggregated sites. The Link Capacity is calculated automatically.

Basics for this widget are addressed on the TX HLD Module.
