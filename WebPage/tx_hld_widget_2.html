Tx - Availability FO

This widget calculates the Availability for a Fiber Optic Link based on the link's general characteristics.

In order to use this widget, perform the following steps:

  • Introduce the FO link characteristics in the Inputs section, according to the given units.
  • Click on the "Calculate" button to generate the FO Link Budget.
  • The FO Link Budget parameters can be found in the Results section.

Basics for this widget are addressed on the TX HLD Module.
